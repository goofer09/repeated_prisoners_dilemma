# A Multi Agent Reinforcement Learning Gym

This repository contains the code and experiments for the MARL Gym project.

## Quick Start

1. Install Pixi.
2. Create the environment:

```bash
pixi install
```

3. Run checks:

```bash
pixi run ty
```

## Project Layout

- `src/final_project/`: Python package source.
- `.github/`: CI workflows.
